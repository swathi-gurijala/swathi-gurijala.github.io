Customer Complaint Classification System using BERT model

Built an intelligent web-based system that classifies customer complaints using the BERT (Bidirectional Encoder Representations from Transformers) model. Users can submit complaints through an interactive interface, and the system automatically categorizes them into relevant classes such as product issues, service delays, or transaction problems. The admin dashboard provides a streamlined view of all complaints, allowing for quick review and action. This project demonstrates the practical application of NLP and transformer-based models in enhancing customer support efficiency and satisfaction.

Chota is a voice-activated chatbot built using HTML, CSS, and JavaScript. It listens to voice commands and can open various Google services such as YouTube, Gmail, or perform searches. With its voice-driven interface and cultural touch, Chota exemplifies how simple web technologies can be combined with voice recognition to create a fun and efficient digital assistant.
